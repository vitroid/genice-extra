# [genice2-extra](https://github.com/vitroid/genice-extra)

Installs all official extra-plugins for [GenIce2](https://github.com/vitroid/GenIce).

version 2.3

## Requirements

* python^3.9
* genice2-cage>=2.3
* genice2-cif>=2.2
* genice2-mdanalysis>=0.7
* genice2-svg>=2.4.1


## Install from PyPI

    % pip install genice2-extra
